# Практическая работа №5
## Основы работы с Git
## Команды Git и их вывод
### git init
Инициализирует репозиторий.
**Вывод:** `Initialized empty Git repository...`
### git add .
Добавляет файлы в отслеживание.
### git commit -m "текст"
Сохраняет изменения.
### git remote add origin <url>
Привязывает удалённый репозиторий.
### git push -u origin master
Отправляет на GitHub.
### git status
Показывает состояние. **Вывод:** `nothing to commit, working tree clean`
### git log
История коммитов.
